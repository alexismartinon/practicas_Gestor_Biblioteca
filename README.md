# 📚 Library Management System / Sistema de Gestión de Biblioteca

## 🇺🇸 English

A library management system developed in Python using Object-Oriented Programming (OOP).
This project allows managing books, users, and lending operations with data persistence.

---

## 🚀 Features

* Book management (add, remove, list)
* User management
* Borrow and return books
* Data persistence using JSON
* Custom exception handling

---

## 🧱 Technologies Used

* Python
* Object-Oriented Programming (OOP)
* JSON (for data storage)

---

## 📂 Project Structure

```
practicas_Gestor_Biblioteca/
│
├── biblioteca.py        # Main library logic
├── libros.py            # Book class and logic
├── usuarios.py          # User management
├── persistencia.py      # Data persistence (JSON)
├── exceptions.py        # Custom exceptions
├── data.py              # Initial or helper data
├── biblioteca.json      # Stored data
├── main.py              # Entry point
├── pyproject.toml       # Project configuration
└── README.md
```

---

## ▶️ How to Run

```bash
python main.py
```
